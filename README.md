# DataPulse

DataPulse is a deterministic CSV and Excel cleaner studio for preparing messy tabular data for analysis.

The project is currently in DP-0001 foundation scope. This first contract establishes the repository, product direction, and application foundations without implementing file upload, parsing, cleaning, exports, reports, or saved history.

## Product Positioning

DataPulse is not an AI cleaning tool. It is designed as a transparent, rule-based cleaning studio where users can inspect issues, choose cleaning rules, preview changes, and export analysis-ready CSV files.

## Current Status

- Repository foundation is being established.
- Backend and frontend foundations are planned in this contract.
- Data cleaning domain contracts are planned in this contract.
- Actual CSV and Excel processing is intentionally deferred.

## Planned Stack

- Backend: Python, FastAPI, Pydantic, Pytest
- Frontend: React, TypeScript, Vite, Vitest
- Future data processing: pandas, openpyxl, xlrd, Python csv module
- Future persistence: SQLite
